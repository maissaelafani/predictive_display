
        PROGRAMME DE SAISIE INTUITIVE

La fonction principale de ce programme est de réaliser une saisie intuitive en proposant 
des prédictions à l'utilisateur lorsque ce dernier écris un message.


        MISE EN PLACE ET INSTALLATION

Pour démarrer le programme il suffira executer la commande "make main" puis "./main".


        INTERFACE ET UTILISATION

Le programme possède une interface graphique dans le terminal.
Pour naviguer entre les différentes fenêtres il faut entrer le chiffre correspondant et appuyer sur entrée.
! BUG CONNU !  - Entrer une lettre au lieu d'un chiffre provoquera un crash 


            FONCTIONNALITÉS

1. Accéder au clavier avec prédiction
FONCTION
Ici l'utilisateur peut saisir du texte et obtenir des predictions.
MODE D'EMPLOI
- Le processus se fait de manière automatique, pas besoin d'appuyer sur 'entrée' à chaque fois, mise à jour des mots prédictives immédiate.
- Le programme est capable de distinguer de nouveaux mots après avoir appuyé sur 'espace'.
- Pour sélectionner une proposition il suffit d'appuyer sur 1,2 ou 3 qui correspondent 
respectivement à la 1ère,2éme et 3éme proposition à partir de la gauche. Le mot sera 
automatiquement remplacé par la proposition choisie.
- Pour quitter la saisie intuitive il faut appuyer sur 'échap'.
! BUG ! - Cette fonctionnalité peut ne pas marcher avec des majuscules et/ou certains caractères speciaux

2. Afficher les mots d'un dictionnaire
FONCTION
on affiche au choix le dictionnaire personnel ou français
MODE D'EMPLOI
il suffit de selectionner dans le menu la fonctionnalité souhaité et appuyer sur 'entrée'

3. Ajouter un mot
FONCTION
Permet d'ajouter un mot dans le dictionnaire personnel
MODE D'EMPLOI
il faut écrire le mot à ajouter et confirmer en appuyant sur 'entrée'

+ 4. Supprimer un mot
FONCTION
Permet de supprimr un mot dans le dictionnaire personnel
MODE D'EMPLOI
Il faut ecrire le mot à supprimer et confirmer en appuyant sur 'entrée'

5. Paramétrer la prédiction
FONCTION
Permet de sélectionner le mode de prédiction :
    -> mode fréquence : tri des propositions selon leur fréquence d'utilisation
    -> mode temporel : tri des proposition selon leur dernière utilisation
MODE D'EMPLOI
Il suffit de selectionner l'option avec son nombre et appuyer sur 'entrée'

6. Quitter le programme
